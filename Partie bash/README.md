# Guide Partie Bash

Tester le projet:

Créer un fichier .gz dans le répertoire courant et appeler ./import-archive.sh nom_du_gz
Fonctionne aussi en appelant plusieurs fichiers .gz dans l'appel
Paramètre -f: force l'import de l'archive (pour un seul fichier)

Supprimer un fichier .gz mis dans le commit initial et appeler ./restore-toolbox.sh pour restaurer le projet
Fonctionne également si on supprime une des lignes dans le .sh-toolbox/archives
